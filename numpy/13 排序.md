### 14. 排序
跟Python内置的列表类型一样，NumPy数组也可以通过sort方法就地排序：
```python
arr = np.random.randn(6)
arr
Output: array([ 0.6095, -0.4938, 1.24 , -0.1357, 1.43 , -0.8469])

arr.sort()
arr
Output: array([-0.8469, -0.4938, -0.1357, 0.6095, 1.24 , 1.43 ])
```
多维数组可以在任何一个轴向上进行排序，只需将轴编号传给sort即可：
```python
arr = np.random.randn(5, 3)
arr
Output: array([[ 0.6033, 1.2636, -0.2555],
                [-0.4457, 0.4684, -0.9616],
                [-1.8245, 0.6254, 1.0229],
                [ 1.1074, 0.0909, -0.3501],
                [ 0.218 , -0.8948, -1.7415]])

arr.sort(1)
arr
Output: array([[-0.2555, 0.6033, 1.2636],
                [-0.9616, -0.4457, 0.4684],
                [-1.8245, 0.6254, 1.0229],
                [-0.3501, 0.0909, 1.1074],
                [-1.7415, -0.8948, 0.218 ]])
```
